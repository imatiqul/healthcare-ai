☁️ **Infra:** 12 Bicep modules, AKS, ArgoCD canary rollouts, KEDA scale-to-zero, zero-trust networking

→ Event-driven architecture — change the event names, keep the infrastructure
→ Module Federation — swap domain MFEs, keep the shell + shared packages
→ AI guardrails pattern applies to triage, fraud detection, recommendations, forecasting
→ Same CI/CD, same observability, same security posture — any domain
